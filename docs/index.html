Are.na Tools and Toys

Playing with the are.na API to make small fun things. A work in progress.

dark

Are.na Hypernormalisation

Pull images and text from are.na and combine them into a strangely familiar documentary critique of late capitalism. Good for full screen ambient displays, or just staring into the void. (h/t to Tom Scott and Dan Williams.)

↗ repo Try it

Are.na Palette

Generate colour palettes from images pulled from your favourite are.na channels. Export as hex, .ase, css, gimp or procreate palettes, or fancy pngs.

↗ repo Try it

Are.na Picks

A homepage that shows random blocks from one or more are.na channels. Lock the ones you want to keep, then shuffle again, or export a zip file of the best combos. Good for inspiration and serendipitous juxtaposition.

↗ repo Try it
